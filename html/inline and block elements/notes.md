
```html
<p>On a starry night, oh my you sleep tight.</p>
<a href="https://www.youtube.com/watch?v=_yTP_L8fC-k&list=RD_yTP_L8fC-k&start_radio=1">MY YOU</a>
````
---
* The `<p>` tag is a **block-level element**

  * Covers full width
  * Always starts on a new line

* The `<a>` tag is an **inline element**

  * Does NOT start a new line
  * Takes only required width
  * Appears in the same line

```css
span {
  display: block;
}

div {
  display: inline;
}
```

:+1: **Block ↔ Inline behavior is NOT fixed** (can be changed using CSS)

---

# Block Elements (Most Commonly Used)

* `<div>`: Generic container for flow content
* `<p>`: Paragraph
* `<h1>` – `<h6>`: Headings
* `<ul>`: Unordered list
* `<ol>`: Ordered list
* `<li>`: List item
* `<form>`: Form container
* `<table>`: Table
* `<section>`: Standalone document section
* `<header>`: Introductory content
* `<footer>`: Footer of a section or page
* `<nav>`: Navigation links
* `<article>`: Self-contained content
* `<aside>`: Side content related to main content
* `<main>`: Main content of document
* `<fieldset>`: Group of form controls
* `<blockquote>`: Quoted block of text
* `<pre>`: Preformatted text
* `<canvas>`: Graphics container
* `<dl>`: Description list
* `<dt>`: Term in description list
* `<dd>`: Description of term
* `<figure>`: Referenced media/content
* `<figcaption>`: Caption for figure
* `<address>`: Contact information
* `<hr>`: Thematic break
* `<tfoot>`: Table footer

# Inline Elements (Most Commonly Used)

* `<a>`: Anchor / hyperlink
* `<img>`: Image
* `<span>`: Generic inline container
* `<input>`: Input field
* `<label>`: Label for form input
* `<strong>`: Strong emphasis
* `<em>`: Emphasized text
* `<br>`: Line break
* `<code>`: Inline code
* `<b>`: Bold text
* `<i>`: Italic text
* `<u>`: Underlined text
* `<small>`: Smaller text
* `<sub>`: Subscript
* `<sup>`: Superscript
* `<mark>`: Highlighted text
* `<q>`: Inline quotation
* `<cite>`: Citation
* `<kbd>`: Keyboard input
* `<samp>`: Sample output
* `<var>`: Variable
* `<time>`: Time/date
* `<abbr>`: Abbreviation
* `<data>`: Machine-readable data
